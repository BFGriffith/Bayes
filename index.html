Bayesian Calculator

Easy drop-in calculator

Download this project as a .zip file Download this project as a tar.gz file

The goal of this project is to provide an easy method for internet authors to add a bayesian calculator to their web pages. This project is still in development, so bugs do exist, if you find any you can log them here

To include any of the following calculators on your page, first you must include the following script tags in the head of your document:

<script src="http://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"></script>
<script src="http://camspiers.github.com/Bayes/javascripts/bayes-calc-so.js" charset="utf-8"></script>

Once these are included on the page, adding a calculator is as easy as pasting the following configuration option of choice into your document:

Simple

<div class="bayes-calc" data-config='{"type": "simple"}'></div>

Simple with graph

<div class="bayes-calc" data-config='{"type": "simple", "graph": true}'></div>

Full with equation

<div class="bayes-calc" data-config='{"type": "full"}'></div>

Full without equation

<div class="bayes-calc" data-config='{"type": "full", "equation": false}'></div>

Full with equation and bar graph

<div class="bayes-calc" data-config='{"type": "full", "graph": true}'></div>

Full with equation and circle graph

<div class="bayes-calc" data-config='{"type": "full", "graph": "circle"}'></div>

Simple a fortiori

<div class="bayes-calc" data-config='{"type": "simple", "afortiori": true}'></div>

Full a fortiori

<div class="bayes-calc" data-config='{"type": "full", "afortiori": true}'></div>

3 Hypotheses

<div class="bayes-calc" data-config='{"type": "full", "num": 3}'></div>
